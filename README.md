# OrangeHRM_Manual_Testing_Project
**Overview**
This project contains manual testing documentation for Demo OrangeHRM. The purpose of this project is to practice and demonstrate QA concepts like functional testing, CRUD operations, search validation and basic workflow testing.

**Project Goal:**
To validate core HR functionalities and improve hands-on experience in manual testing, test case design & execution.

**Modules Tested**
Authentication
PIM (Employee Management)
Recruitment
Leave Management
Directory
